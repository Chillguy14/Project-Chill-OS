# IcyOS 🧊

IcyOS is my personal Linux project, built from the ground up.  
The goal is to create a fast, simple, and blueish OS with Hyprland, Waybar, and custom configs.

## 🔹 Features
- Hyprland Wayland environment
- Custom Waybar with personalized widgets
- Kitty terminal with themes
- Dotfiles and scripts for quick setup

## 🔹 Installation
1. Download the ISO (coming soon)
2. Boot from USB
3. Follow the scripts in `setup.sh` to install everything automatically

## 🔹 How to Contribute
1. Fork the repository
2. Create a branch with your changes
3. Submit a pull request with a description of what you changed
